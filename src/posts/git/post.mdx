I checked my history of commands ran in past 3 days and 90% of these commands are just git commands. No Wonder you always need to have your own alias and feel like THAT IT GUY

## Git Aliases

First thing that you can optimize is this. Create some aliases

<CodeDivWrapper>

```bash
git config --global alias.a "!git add ."
git config --global alias.c "!git commit -m 'auto generated message' "
git config --global alias.p "!git push origin $(git rev-parse --abbrev-ref HEAD)"
git config --global alias.ac "!git a; git c"
git config --global alias.acp "!git ac; git p"
```
