## Adding environment variables

<CodeDivWrapper>

```bash
set -x VAR VAL

# For setting a variable universally
set -Ux FOO bar

# Sample
set -x ANDROID_HOME $HOME/Library/Android/sdk
echo $ANDROID_HOME
```

</CodeDivWrapper>

## Setting path variable

<CodeDivWrapper>

```bash
set -U fish_user_paths PATH_TO_BIN $fish_user_paths

# Sample
set -U fish_user_paths /Applications/Contents/jre/jdk/Contents/Home/bin $fish_user_paths
```

</CodeDivWrapper>

## Alias

Make surr you add these in your fish config file (~/.config/fish/config.fish) so that it remains after you restart your computer

<CodeDivWrapper>

```bash
alias g "git"
alias gacp "git acp"
alias gac "git ac"
alias ni "npm install"
alias na "npm install --save"
alias nad "npm install --save-dev"
alias yi "yarn"
alias ya "yarn add"
```

</CodeDivWrapper>
