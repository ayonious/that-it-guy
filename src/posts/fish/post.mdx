## Installation

Through brew. Install brew first

<CodeDivWrapper>

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
brew install fish
```

</CodeDivWrapper>

Add this line `eval "$(/opt/homebrew/bin/brew shellenv)"` in fishConfig file to run each time you open fish terminal


<CodeDivWrapper>

```bash
sudo nano ~/.config/fish/config.fish
```

</CodeDivWrapper>

## Make fish default shell

### Step1

Find where is fish executable installed after homebrew installation.

<CodeDivWrapper>

```bash
which fish
```

</CodeDivWrapper>

For me the location is `/opt/homebrew/bin/fish`

### Step2

Put the location at end of `/etc/shells`

<CodeDivWrapper>

```bash
sudo nano /etc/shells
```

</CodeDivWrapper>

### Step3

Run chsh with this location

<CodeDivWrapper>

```bash
chsh -s /opt/homebrew/bin/fish
```

</CodeDivWrapper>

## Adding environment variables

<CodeDivWrapper>

```bash
set -x VAR VAL

# For setting a variable universally
set -Ux FOO bar

# Sample
set -x ANDROID_HOME $HOME/Library/Android/sdk
echo $ANDROID_HOME
```

</CodeDivWrapper>

## Setting path variable

<CodeDivWrapper>

```bash
set -U fish_user_paths PATH_TO_BIN $fish_user_paths

# Sample
set -U fish_user_paths /Applications/Contents/jre/jdk/Contents/Home/bin $fish_user_paths
```

</CodeDivWrapper>

## Installing nvm with fish

Install fisher `https://github.com/jorgebucaran/fisher`

<CodeDivWrapper>

```bash
curl -sL https://git.io/fisher | source && fisher install jorgebucaran/fisher
```

</CodeDivWrapper>


Now install bass

<CodeDivWrapper>

```bash
fisher install edc/bass
```

</CodeDivWrapper>

Add this to `~/.config/fish/config.fish` file

<CodeDivWrapper>

```bash
function nvm
    bass source ~/.nvm/nvm.sh --no-use ';' nvm $argv
end
```

</CodeDivWrapper>

## Alias

Make sure you add these in your fish config file (~/.config/fish/config.fish) so that it remains after you restart your computer

<CodeDivWrapper>

```bash
alias g "git"
alias gac "g status; g add .; g commit -m 'test message'"
alias gacp "gac; git push -u origin HEAD"
```

</CodeDivWrapper>
