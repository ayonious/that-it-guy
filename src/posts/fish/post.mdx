Writing the article about git alias gave me a better idea of making alias of everything. Including all my tools like npm and yarn. So here goes my next list

## Alias

<CodeDivWrapper>

<Highlight>

```bash
alias g "git"
alias gacp "git acp"
alias gac "git ac"
alias ni "npm install"
alias na "npm install --save"
alias nad "npm install --save-dev"
alias yi "yarn"
alias ya "yarn add"
```

</Highlight>

</CodeDivWrapper>
