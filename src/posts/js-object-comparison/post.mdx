# What options can we think about?

```js
const a = { a: 1, b: 2, c: { a: 1 } };
const b = { a: 1, b: 2, c: { a: 1 } };
const c = { b: 2, a: 1, c: { a: 1 } };
a === b; // false
a == b; // false
JSON.stringify(a) === JSON.stringify(b); // true
JSON.stringify(a) === JSON.stringify(c); // false
```
