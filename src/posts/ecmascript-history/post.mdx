# ECMAscript 2015 (ES6)

## Multi lines Strings

Remember this nightmare?

<Highlight>

</Highlight>

</CodeDivWrapper>

<CodeDivWrapper>

  some joke,

# ECMAscript 2018

## Spread operator

<CodeDivWrapper>

<Highlight>

```js
const person = {
  name: 'asdf',
  age: 12,
}
const personAgain = {
  school: 'some good one'
  ...person
} // {school: 'some good one', name: 'asdf', age: 12, }
```

</Highlight>

</CodeDivWrapper>

<CodeDivWrapper>

<Highlight>

```js
const person = {
  name: "asdf",
  age: 12,
  school: "no school",
};
const { name, ...others } = person;
// others = { age: 12, school: 'no school' }
```

</Highlight>

</CodeDivWrapper>
