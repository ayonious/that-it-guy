Renovate is THE nicest thing I did for my opensource projects. You don't really want to manually keep updating all your repositories. Let this Renovate BOT do this trick

## Installation Steps

1.Config file (example from my projects)

<CodeDivWrapper>

<Highlight>

```json
{
  "extends": ["config:base"],
  "rangeStrategy": "bump",
  "bumpVersion": "patch",
  "automerge": true,
  "lockFileMaintenance": {
    "enabled": true
  }
}
```

</Highlight>

</CodeDivWrapper>

2.Goto renovate bot and allow to make prs in that repo
