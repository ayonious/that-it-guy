# ECMAscript 2019 (ES10)

## Array.flatMap() Array.Flat()

<CodeDivWrapper>

<Highlight>

```js
const a = [1, 2, 3];
a.map((v1) => [v1, v1 * 3]); // [ [1,3], [2,6], [3,9]]
a.flatMap((v1) => [v1, v1 * 3]); // [ 1, 3, 2, 6, 3, 9]
```

</Highlight>

</CodeDivWrapper>

<CodeDivWrapper>

<Highlight>

```js
const a = [1, [2, 3], [4, 5, [6, 7]]];
a.flat(); // [1, 2, 3, 4 , 5, [2,6]]
a.flat().flat(); // [1, 2, 3, 4, 5, 2, 6]
a.flat(Infinity); // [1, 2, 3, 4, 5, 2, 6]
```

</Highlight>

</CodeDivWrapper>
